# Support

MobSec Studio is currently a beta desktop application.

Use GitHub Issues for:

- Reproducible bugs.
- Platform-specific problems.
- Feature requests.
- Documentation gaps.

Before opening an issue, include:

- Operating system and version.
- MobSec Studio version.
- Android device/emulator model and API level when relevant.
- Whether the device is rooted.
- ADB, Frida, JADX, proxy, or logcat logs when relevant.
- Clear reproduction steps.

For private security reports, use the process in `SECURITY.md`.
