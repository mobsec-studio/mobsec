# Code Of Conduct

MobSec Studio is built for serious security work, and the community should feel
professional, focused, and respectful.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but be precise when discussing risk.
- Keep feedback technical, actionable, and kind.
- Respect boundaries around responsible security research.
- Help keep public discussions safe for maintainers, contributors, and users.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing private information without consent.
- Encouraging unauthorized access, credential theft, persistence, or stealthy
  misuse.
- Posting exploit details for third-party targets without authorization.
- Repeatedly derailing technical discussions.

## Enforcement

Maintainers may edit, hide, or remove comments, close issues, block users, or
take other reasonable moderation actions to protect the project and community.
